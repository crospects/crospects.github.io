C-Shore, an OS for all!

C-Shore is a simple, compact OS built on GNODE and openSUSE. Above you can see a screenshot inside of C-Shore with the VLC Media Player application open. You can see that the UI is similar to other Linux distributions.


Currently C-Shore is not available for download, as it is still under developement. When it is, instructions on how to download, run and test this operating system on your computer will be posted here.
